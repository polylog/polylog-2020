13 и 14 декабря в Москве в Коворкинге Нагатино впервые прошел Хакатон Apps4all. За два дня непрерывного кодинга были созданы прототипы приложений, большая часть из которых может принести не один миллион долларов. Организаторами мероприятия выступили Консалтинговая Группа «Полилог» и Компания «Apps4all».

Участников хакатона (программистов, дизайнеров, менеджеров продуктов, предпринимателей) консультировали специалисты по открытым данным, представители компаний-партнеров, менторы, инвесторы и эксперты рынка.

Мероприятие поддержали такие крупнейшие мировые производители высокотехнологичной продукции и программного обеспечения, как: Intel, Google, Microsoft, Samsung. Каждая из компаний использовала свои критерии отбора. Завершилось мероприятие демонстрацией разработанных прототипов. Оценивало реализованные проекты жюри, в состав которого вошли отраслевые эксперты. Участники боролись за победу в пяти номинациях.

Приложение для Androidx86

Intel (номинация «Приложение для Androidx86») наградила своих лауреатов Harris Beach Developer Ultrabook, которого еще даже нет в продаже. Технологию для разработки приложения можно было выбрать самому — HTML5, Android SDK или NDK. Главное условие – оригинальность идеи. Приз за «Самое инновационное Androidx86 приложение» достался каждому участнику команды. Кроме того, авторы самых интересных проектов получили возможность дальнейшего сотрудничества с компанией Intel: инженерное сопровождение проекта и маркетинговую поддержку.

Приложения с использованием технологий Google

Для Google (номинация «Приложения с использованием технологий Google»), главными критериями для победы были: полезность для пользователя, потенциальная популярность и перспективность, технологичность и дизайн, креативное использование технологий Google и открытых технологий и стандартов.

Мобильное приложение с бэкендом в Windows Azure

Компания Microsoft (номинация «Мобильное приложение с бэкендом в Windows Azure») предложила участникам ресурсы Windows Azure бесплатно по программе для стартапов Microsoft BizSpark и консультацию по его использованию. Windows Azure – это надежная масштабируемая инфраструктура, единое облачное хранилище данных, авторизация пользователей через социальные сети и многое другое. Первые 2 команды получили от компании xBox One, а занявшая 3 место команда выиграла 3 смартфона Nokia Lumia 625 от компании Nokia. Счастливыми призерами в трех номинациях стали авторы проекта под названием Logos, работавшие под лозунгом “because we care”.

Мобильное приложение с использованием Samsung Mobile SDK

Следующая номинация — «Мобильное приложение с использованием Samsung Mobile SDK» от компании Samsung. Samsung Mobile SDK для Android включает в себя 10 независимых пакетов. Эти инструменты можно использовать совместно или отдельно, они позволяют ускорить и облегчить разработку приложений для Android. Победителей ждали смартфоны от Samsung. В данный номинации выиграл проект Planch — сервис для планирования проектов, загруженности команды и учета времени.

Мобильное приложение на платформе Tizen

Победителей номинации «Мобильное приложение на платформе Tizen» ждали сэмплы Tizen. Участники должны были создать одно из первых приложений для платформы Tizen, которая скоро выходит на рынок. Призерами стали авторы проекта Свобода речи.

Кроме того, участники смогли принять участие в образовательной программе. Мероприятие помогло им, как проявить свои способности в борьбе за призы, так и обменяться опытом с коллегами, завести новые контакты и приобрести ценные знания. Во время хакатона действовала и ярмарка вакансий, в рамках которой ведущие HR-агентства — HeadHunter, DigitalHR, ITMozgFuture Today — представили обзор перспективных вакансий IТ-рынка.

Официальными партнёрами мероприятия так же выступили компании: Cloud Castle, ABBYY, Манн, Иванов и Фербер, типичный программист, Lamoda.ru, Белая Дача, Пятый океан, The Wind Hostel, Ква-ква парк.

Хакатон, организованный Apps4all, имеет большое значение для развития отечественного рынка мобильных телекоммуникаций, выявляя и стимулируя талантливых авторов и разработчиков мобильных приложений. Хакатон Apps4all – это место, где креативные, инновационные идеи воплощаются в конкретные разработки и решения, полезные для потребителей.
